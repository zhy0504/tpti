# TPTI 结果类型 与 对应题号 / 对应知识索引

## 说明

- 当前项目中的结果类型，并不是由单独某一题直接决定，而是由**总分 + 主要失分维度**共同决定
- 因此，下表中的"对应题号"本质上表示：**哪些题目所在维度，会显著影响该结果类型的出现概率**
- 可用于策划、运营、内容标注、结果页解释文案整理

## 结果类型 与 对应题号 / 对应知识

| 结果类型 | 对应题号 | 对应知识 |
|---|---|---|
| **【金钟罩体质】** `action_guard` | q3、q4、q6、q11、q20、q21、q22、q23；q8、q9、q27、q28、q29、q30 | 症状识别是否及时；出现持续咳嗽咳痰后是否知道应尽快检查；是否具备正确就医行动；是否理解规范治疗、不能擅自停药；是否能识别常见治疗误区 |
| **【嘴强王者体质】** `knowledge_online` | q3、q4、q6、q11、q20、q21、q22、q23；q8、q9、q27、q28、q29、q30 | 整体知识储备较好，但在**症状识别/就医行动**或**规范治疗/常见误区**上仍有明显失分；说明知道不少，但行动判断或治疗认知还不够 |
| **【养成系体质】** `steady_learner` | q1、q7、q12、q13、q14、q15；q8、q9、q27、q28、q29、q30；以及其他分散失分题 | 基础认知已具备，但在传播、症状、预防、治疗等细节上还不够稳定；错题通常较分散，没有特别明显的单一爆雷维度 |
| **【精准踩雷体质】** `pitfall_tripper` | q2、q16、q17、q18、q19；q5、q10、q24、q25、q26 | 容易在**传播途径**和**预防方法**上被误导；对飞沫传播、通风、咳嗽礼仪、日常防护等核心知识点理解不稳，容易踩常见误区 |
| **【忍者硬抗体质】** `delay_observer` | q3、q4、q6、q11、q20、q21、q22、q23 | 主要问题集中在症状识别和行动时机；知道一些知识，但面对持续咳嗽、咳痰、低热、盗汗等信号时，容易拖延、观望，不能及时就医检查 |
| **【脆皮裸奔体质】** `need_recharge` | q1～q30（全局题库） | 基础知识整体偏弱；传播方式、症状判断、及时就医、规范治疗、预防动作等多个维度都存在明显短板，需要系统补齐 |

## 结果类型与维度关系简表

| 结果类型 | 主要关联维度 | 说明 |
|---|---|---|
| **【金钟罩体质】** `action_guard` | `symptom_action`、`treatment_misunderstanding` | 高分且这两个关键维度失分不多，说明既懂也会做 |
| **【嘴强王者体质】** `knowledge_online` | `symptom_action`、`treatment_misunderstanding` | 高分，但在行动判断或治疗误区上仍有明显漏洞 |
| **【养成系体质】** `steady_learner` | 多维度分散 | 有基础，但细节还没完全补齐 |
| **【精准踩雷体质】** `pitfall_tripper` | `transmission`、`prevention` | 传播和预防相关误区较多 |
| **【忍者硬抗体质】** `delay_observer` | `symptom_action` | 最大问题是识别不够快、行动不够及时 |
| **【脆皮裸奔体质】** `need_recharge` | 全维度 | 多个维度都存在明显短板 |
