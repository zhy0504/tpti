# TPTI 结果类型 → 对应题号 → 对应知识点

## 说明

- 当前项目中的结果类型，并不是由单独某一题直接决定，而是由 **总分 + 主要失分维度** 共同决定。
- 因此，下表中的“对应题号”本质上表示：**哪些题目所在维度，会显著影响该结果类型的出现**。
- 当前运行时抽题规则为：**每轮固定 10 题，每个维度 2 题**。
- 题库已经扩充并去重，单个知识点不再被大量重复提问；更适合从“维度覆盖”角度理解结果页，而不是把结果类型理解成某几道固定题。

## 当前题库的维度分布（40 题题库）

| 维度 | 对应题号 | 覆盖重点 |
|---|---|---|
| `knowledge_basic` | q1～q8 | 病原体、最常见累及部位、易感人群、卡介苗作用、结核病可防可治、潜伏感染与检测结果基础认知 |
| `transmission` | q9～q16 | 飞沫传播、非主要传播方式、密闭空间风险、活动性肺结核传播、传播环境判断 |
| `symptom_action` | q17～q24 | 咳嗽咳痰 2 周警戒线、常见症状、密切接触后处理、及时检查、检测阳性后的进一步评估 |
| `prevention` | q25～q32 | 通风、咳嗽礼仪、多人室内环境防护、陪诊防护、重点人群筛查、预防活动性结核 |
| `treatment_misunderstanding` | q33～q40 | 规范治疗、不能自行停药、漏服与耐药风险、副作用应对、复查、偏方误区、潜伏感染规范处理 |

## 结果类型 → 主要关联题号范围 → 对应知识点

| 结果类型 | 主要关联题号 | 对应知识点 |
|---|---|---|
| **【金钟罩体质】** `action_guard` | q17～q24；q33～q40 | 总分较高，且症状识别 / 就医行动与治疗误区这两个关键维度合计失分较少，说明既懂也会做 |

### 5. 代码层面的判定关系

#### 8-10 分 → 结核病防治知识达人

可能类型只有：

- `action_guard`
- `knowledge_online`

判定逻辑：

- 如果 **症状识别 / 就医行动** 与 **治疗误区** 这两个行动相关维度合计失分达到 2 题及以上
  → `knowledge_online`
- 否则
  → `action_guard`

#### 6-7 分 → 进阶选手

可能类型只有：

- `steady_learner`
- `delay_observer`

判定逻辑：

- 如果错题最多的维度是 `symptom_action`
  → `delay_observer`
- 否则
  → `steady_learner`

#### 3-5 分 → 还需补课

可能类型只有：

- `steady_learner`
- `pitfall_tripper`
- `delay_observer`

判定逻辑：

- 如果错题最多的维度是 `transmission` 或 `prevention`
  → `pitfall_tripper`
- 如果错题最多的维度是 `symptom_action`
  → `delay_observer`
- 否则
  → `steady_learner`

#### 0-2 分 → 建议立刻知识充电

可能类型只有：

- `need_recharge`
