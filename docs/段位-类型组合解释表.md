# TPTI 知识水平 × 类型组合解释表

## 说明

- **知识水平**：按总分划分，表示整体答题水平。
- **类型**：按失分特征划分，表示主要短板和知识 / 行动画像。
- 因此，知识水平和类型不是同一层信息，也不是所有组合都能出现。
- 当前项目中，理论上有 4 个知识水平 × 6 个类型 = 24 种组合，但按代码判定逻辑，实际只有 8 种有效组合。

## 五、代码层面的判定关系

### 1. 8-10 分 → 结核病防治知识达人

可能类型只有：

- `action_guard`
- `knowledge_online`

判定逻辑：

- 如果 **症状识别 / 就医行动** 与 **治疗误区** 这两个行动相关维度合计失分达到 2 题及以上
  → `knowledge_online`
- 否则
  → `action_guard`

### 2. 6-7 分 → 进阶选手

可能类型只有：

- `steady_learner`
- `delay_observer`

判定逻辑：

- 如果错题最多的维度是 `symptom_action`
  → `delay_observer`
- 否则
  → `steady_learner`

### 3. 3-5 分 → 还需补课

可能类型只有：

- `steady_learner`
- `pitfall_tripper`
- `delay_observer`

判定逻辑：

- 如果错题最多的维度是 `transmission` 或 `prevention`
  → `pitfall_tripper`
- 如果错题最多的维度是 `symptom_action`
  → `delay_observer`
- 否则
  → `steady_learner`

### 4. 0-2 分 → 建议立刻知识充电

可能类型只有：

- `need_recharge`

## 七、备注

- 当前结果页展示方式为：**结果类型 + 当前知识水平 + 标签 + 重点建议**。
- “类型”回答的是：**你更像哪一类人、主要短板是什么**。
- “知识水平”回答的是：**你总体答题水平处于哪个区间**。
- 两者组合后，才构成完整的结果解释。
